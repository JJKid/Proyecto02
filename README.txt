Ejecución del programa:

1. Instalar python3
2. Instalar biblioteca Pillow (mas referencia https://pillow.readthedocs.io/en/stable/installation.html)

	python3 -m pip install --upgrade pip
	python3 -m pip install --upgrade Pillow

3. Para llevar a cabo el procedimiento de ocultar o develar:
Moverse a la carpeta /src y ejecutar: python .\steganography.py 

4. Seguir las instrucciones en pantalla. Teniendo cuidado con poner la extensión de los
nombres de archivos indicada en cada paso

Los archivos de entrada (imagen original y texto a ocultar) estan en la carpeta /data
y los archivos de salida se guardan en la carpeta /out
